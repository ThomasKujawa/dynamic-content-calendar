# Dynamischer Inhaltskalender

Dieser dynamische Inhaltskalender hilft dabei, verschiedene Inhalte für Ihre Plattform zu planen und zu organisieren. Er bietet Benutzern die Möglichkeit, Inhalte einfach zu verwalten und sicherzustellen, dass alle geplanten Inhalte rechtzeitig veröffentlicht werden.

## Funktionen
- **Benutzerfreundliche Oberfläche:** Der Kalender ist leicht verständlich und ermöglicht eine einfache Navigation.
- **Planung von Inhalten:** Nutzer können zukünftige Inhalte planen und festlegen, wann diese veröffentlicht werden sollen.
- **Farbcodierung:** Inhalte können nach Typ oder Status farblich codiert werden, um eine schnellere Übersicht zu ermöglichen.
- **Benachrichtigungen:** Automatische Erinnerungen für bevorstehende Veröffentlichungen.

## Installation
Um den dynamischen Inhaltskalender zu installieren, folgen Sie diesen Schritten:
1. Klonen Sie das Repository: `git clone https://github.com/ThomasKujawa/dynamic-content-calendar.git`
2. Installieren Sie die Abhängigkeiten:
   ```bash
   cd dynamic-content-calendar
   npm install
   ```
3. Starten Sie die Anwendung:
   ```bash
   npm start
   ```

## Nutzung
Nach dem Start der Anwendung können Sie Inhalte hinzufügen, bearbeiten oder löschen, indem Sie die Optionen im Menü verwenden. Der Kalender zeigt eine übersichtliche Ansicht aller geplanten Inhalte, sodass Sie auf einen Blick sehen können, was als Nächstes kommt.
